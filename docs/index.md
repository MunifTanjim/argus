## Get started

Install on macOS or Linux, wire up hooks, and watch:

```sh
# install the latest release
curl -fsSL https://argus.muniftanjim.dev/install.sh | bash

# let Argus track each session's status
argus hooks install

# open the TUI
argus
```

Prefer Go? `go install github.com/MunifTanjim/argus/cmd/argus@latest` — or follow the full [Installation](/getting-started/installation) guide.
